# Flappy Bird Game (Python)

## Overview

This project is a Flappy Bird clone developed in **Python** using the **Pygame** library as part of my Information and Communication Technology (ICT) coursework at **Air University**.

The game recreates the classic Flappy Bird experience, where the player controls a bird that must fly through continuously generated obstacles without colliding with pipes or the ground. This project helped me understand the fundamentals of game development, event handling, collision detection, and graphical user interface programming in Python.

---

## Features

- Gravity-based bird movement
- Smooth jump mechanics
- Randomly generated moving pipes
- Collision detection with obstacles and the ground
- Real-time score tracking
- Game Over screen
- Restart functionality
- Background graphics and game assets using Pygame

## Installation

```bash
git clone https://github.com/ManalYasir/Flappy-Bird-Game-Python.git
```

### Install the required library

```bash
pip install pygame
```

### Run the game

```bash
python main.py
```

---

## Gameplay

The player controls a bird that continuously falls due to gravity. Pressing the jump key causes the bird to flap upward. The objective is to pass through the gaps between pipes without colliding with obstacles or the ground. The score increases as the player successfully passes each set of pipes.
